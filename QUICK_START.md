# 🚀 快速开始 - 5分钟测试指南

## Windows用户

1. **双击运行** `start.bat`
2. 等待浏览器自动打开,或手动访问显示的地址
3. 在手机上输入显示的"手机访问"地址

## Mac/Linux用户

1. **在终端运行**:
   ```bash
   cd step_counter_app
   ./start.sh
   ```
2. 访问显示的地址

## 测试步骤

### 在电脑浏览器测试界面

1. 打开 `http://localhost:5000`
2. 查看界面是否正常显示
3. 点击"重置今日"测试按钮功能

### 在手机上测试传感器

1. **确保手机和电脑在同一WiFi**
2. 在手机浏览器输入电脑IP地址 (脚本会显示)
   例如: `http://192.168.1.100:5000`
3. 点击"开始记录"
4. **拿着手机走动**,观察步数变化

### iOS特别说明

如果看到"需要授权"提示:
1. 点击"授予权限"按钮
2. 在弹出窗口选择"允许"
3. 再次点击"开始记录"

### 测试功能清单

✅ 页面能否正常打开
✅ "开始记录"按钮是否工作
✅ 走动时步数是否增加
✅ 卡路里和距离是否自动计算
✅ 进度环是否更新
✅ "重置今日"按钮是否有效
✅ 历史记录是否保存

## 常见测试问题

### 问题1: 手机访问不到
- 检查手机和电脑是否在同一WiFi
- 检查防火墙是否允许5000端口
- 尝试关闭电脑防火墙测试

### 问题2: 步数不增加
- 确认已点击"开始记录"
- iOS用户确认已授予权限
- 尝试更用力地走动
- 查看浏览器控制台是否有错误

### 问题3: 页面打不开
- 确认Python已正确安装
- 确认依赖包已安装
- 查看命令行是否有错误信息
